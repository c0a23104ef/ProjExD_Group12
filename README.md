# **KoukaTale**

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんからの攻撃を耐えるて、こうかとんを倒す。

## ゲームの遊び方
* プレイヤーは「こうげき」「こうどう」「アイテム」「(みのがす)」の4つから行動を選ぶ
* 矢印でハートを操作し、こうかとんからの攻撃を避ける
* こうかとんのHPが0になるまたは、こうかとんが疲れたらゲーム終了となる
* 自分のHPが0になったらGameOverになる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 担当追加機能
* ビーム(上下左右から):田村
* 横から流れる障害物:興
    + 予告するビームとそこに出てくるビームを出す！
* 画面内で玉がバウンド:順
* 夢の卵攻撃:杉本
* 地雷：石川
